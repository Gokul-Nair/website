Malayalam Font Embed Bookmarklet

This is a set of bookmarklets that allows you to read the pages in your favourite fonts even if the system you are using does not have them.

Drag and drop any of the following links to the bookmarks tool of your browser(or add to the favourites). When you are in a page, click on that link in bookmarks toolbar or favourites menu to get the page in the given font.

  • Rachana
  • Meera
  • AnjaliOldLipi
  • Suruma
  • Kalyani
  • Raghu Malayalam
  • Dyuthi
  • Lohit Malayalam
